Initial setup of the Raspberry Pi out of the box.

Once the default Raspbian Wheezy image installed, I use the following steps to get the basic software I need and configuration that works well for me. I do all of this with an SSH connection, I never hooked up a display nor keyboard to the Raspberry Pi.

My 10 step guide :

  1. Do the initial configuration with the provided utility

    sudo raspi-config

  2. Update all software to latest versions

    sudo apt-get update
    sudo apt-get upgrade
    sudo apt-get dist-upgrade

  3. Change “raspberry” hostname by editing following 2 files

    sudo nano /etc/hostname
    sudo nano /etc/hosts
    reboot

  4. Change passwords

    >set root user password :
    sudo passwd

    >Change pi password :
    su
    passwd pi

  5. Check serial / proc speed

    cat /proc/cpuinfo

  6. Install Remote Desktop (remote login into graphical mode)

    sudo apt-get install xrdp

    > You can now remotely login into the graphical raspberry pi interface with “Remote Dektop” and use the built in WIFI utility to setup the wifi connection !

    cf : http://www.raspberrypiblog.com/2012/10/how-to-setup-remote-desktop-from.html

  7. Install Lighttpd (webserver) with python binding

    sudo apt-get install lighttpd

    > Change the directory owner and group
    sudo chown www-data:www-data /var/www

    > Allow the group to write to the directory
    sudo chmod 775 /var/www

    > Add the pi user to the www-data group
    sudo usermod -a -G www-data pi

    > Edit lighttpd config file :
    sudo nano /etc/lighttpd/lighttpd.conf

    >Change/add bold statements :

    server.modules = (
            “mod_access”,
            “mod_alias”,
            "mod_fastcgi",
            “mod_compress”,
            “mod_redirect”,
    )

    index-file.names            = ( “index.php”, "index.py", …
    static-file.exclude-extensions = ( “.php”, ".py", “.pl”, “.fcgi” )

    Add at end :

    $HTTP[“url”] =~ “^/” {
            cgi.assign = ( “.py” => “/usr/bin/python” )
    }

    SAVE !

    > Copy config file :

    sudo cp /etc/lighttpd/conf-available/10-cgi.conf /etc/lighttpd/conf-enabled

  8. Allow lighttpd usr (ww-data) to have root access
    (security risk but needed for GPIO access)

    sudo visudo

    > Add at end :

    www-data ALL=(ALL) NOPASSWD: ALL

    > SAVE and restart web server !

    sudo service lighttpd force-reload

  9. Set Python to execute as root
    (Security risk ! but allows easy access to GPIO)

    cd /usr/bin
    sudo chmod u+s python

  10. Add some needed python libraries and update to latest versions

    sudo apt-get install python-daemon
    sudo apt-get install python-dev
    sudo apt-get install python-pip
    sudo easy_install -U distribute

That’s it. Raedy to do some programming ;-)
